## 🎯 Краткий обзор

| Параметр         | Значение                                            |
|------------------|-----------------------------------------------------|
| **Жанр**         | Карточная экономическая стратегия с engine-building |

### 🌍 Сеттинг
> Мир после «Великого Гава», где собаки управляют университетами. Главный ресурс — **Корм**: валюта, политический вес и мотивация команды. Игроки строят «движок» генерации ресурсов, зависящий от случайного цвета раунда.

## 🎮 Как играть

### 🔄 Базовый цикл хода
```
1. Начало раунда → Генерация активного цвета
2. Фаза покупки → Приобретение карт с рынка (1 карта/ход)
3. Фаза активации → Карты цвета раунда срабатывают автоматически
4. Завершение хода → Передача хода следующему игроку
```

### 🎨 Цветовая система

| Цвет          | Сектор           | Эффект                                    | Частота |
|---------------|------------------|-------------------------------------------|---------|
| 🔵 **Blue**   | Производственный | Доход **всем** игрокам с такой картой     | 45.5%   |
| 🟡 **Gold**   | Коммерческий     | Личный доход из банка                     | 27.3%   |
| 🔴 **Red**    | Теневой          | Кража монет у случайного оппонента        | 22.7%   |
| 🟣 **Purple** | Интриги          | Спецэффекты: кража карт, изменение правил | 4.5%    |

### 🏆 Условия победы
- **Основное**: Первый игрок с **≥100 корма**
- **Альтернатива**: Максимальный баланс после 20 раундов (если никто не достиг цели)

## 🃏 Система карт

### 📦 Структура карты
```csharp
public class Card {
    public string Name { get; }      // Уникальное название
    public CardColor Color { get; }  // Цвет/сектор
    public string Effect { get; }    // DSL-команда: "GET 3", "STEAL_MONEY 2 RANDOM"
    public int Cost { get; }         // Цена покупки
    public int Reward { get; }       // Базовый эффект
    public int Weight { get; }       // Вес вероятности появления (1-100)
}
```

### 🧩 Примеры карт

| Название        | Цвет   | Cost | Эффект                 | Описание                                  |
|-----------------|--------|------|------------------------|-------------------------------------------|
| 🔵 Кофе-брейк   | Blue   | 1    | `GETALL 1`             | +1 корм всем игрокам с этой картой        |
| 🟡 Стипендия    | Gold   | 1    | `GET 1`                | +1 корм только вам                        |
| 🔴 Аудит кормов | Red    | 2    | `STEAL_MONEY RANDOM 2` | Украсть 2 корма у случайного оппонента    |
| 🟣 Плагиат      | Purple | 8    | `STEAL_CARD`           | Украсть случайную карту из руки оппонента |

## ⚙️ Механики и баланс

### 🎲 Генерация активного цвета
Цвет раунда определяется взвешенным рандомом:
```csharp
// Из GameSettings.cs
ColorChanceBlue = 50,    // 45.45%
ColorChanceGold = 30,    // 27.27%
ColorChanceRed = 25,     // 22.73%
ColorChancePurple = 5,   // 4.55%
// Σ = 110
```

### 💜 Механика «Любимый цвет»
Автоматически определяется по составу вашей руки и даёт пассивные бонусы:

```
🔵 > 🔴 > 🟡 > 🟣 > 🔵  (цикл не транзитивности)
```

| Любимый цвет  | Бонусы                                    | Уязвимости                       |
|---------------|-------------------------------------------|----------------------------------|
| **🔵 Blue**   | Доход от всех синих карт                  | Красные крадут в 2× больше       |
| **🟡 Gold**   | Блокирует 50% краж от красных             | Приоритетная цель для фиолетовых |
| **🔴 Red**    | Монополия на синие (доход только вам)     | Штраф -50% к золотым картам      |
| **🟣 Purple** | +50% к золотому доходу, охотник на золото | Не получает доход от чужих синих |

> 💡 **Дизайн-решение**: Фиолетовый любимец активируется с вероятностью ~0.1% — это «легендарная» стратегия с высоким риском и наградой.

### ⚖️ Баланс конфронтации
Ожидаемый эффект Red-карты «Аудит кормов» (Cost=2, Yield=2):
```
E[Доход] = P(Red) × P(на рынке) × P(покупка) × Yield × P(успех)
         = 0.227 × 0.0625 × 0.8 × 2 × 0.95 ≈ 0.0216 монет/раунд
         
→ За 12 раундов: ~0.26 монет ожидаемого дохода
→ Низкий математический вклад, но высокий психологический эффект ✅
```

## 📈 Прогрессия и контент

### 📊 Диаграмма вознаграждения
```
Мощность игрока (доход/ход)
    ↑
100 │                              ╭───🏆 ПОБЕДА
    │                        ╭────╯
 75 │                  ╭────╯     ← «Золотая середина»
    │            ╭────╯
 50 │      ╭────╯               ← Рост синергий
    │ ╭────╯
 25 │╭─╯                        ← Обучение механике
    ╰─────────────────→ Раунды
      1   5   10  15  20
```

### 🎯 Три этапа игры

| Этап           | Раунды | Доступные карты      | Фокус геймплея              |
|----------------|--------|----------------------|-----------------------------|
| 🟢 **Стартап** | 1–5    | 3 карты (Cost 1)     | Обучение, первый движок     |
| 🟡 **Рост**    | 6–10   | +4 карты (Cost 2-3)  | P2P-взаимодействия, тактика |
| 🔴 **Финал**   | 11–20  | +17 карт (Cost 4-20) | Комбо, гонка за победой     |

### 🔄 Адаптивный рынок
Карты появляются на рынке только если `Card.Cost <= текущий_раунд`:
```
Раунд 1 → Карты за 1 корм
Раунд 5 → Карты за 1-5 кормов  
Раунд 10 → Все карты до 10 кормов
```
> Это обеспечивает естественную прогрессию и предотвращает «слом» баланса ранней покупкой сильных карт.

### 🧪 Балансировка методом Монте-Карло
Сервис `CardProbabilityCalculator` запускается при старте сервера и выводит в консоль:
- Вероятности появления и активации каждой карты
- Ожидаемую окупаемость (Cost / Expected Yield)
- Рекомендации по балансу

**Пример вывода**:
```
🔵 Blue   | Карт: 180 | Ср. окупаемость: 11.0 ходов | Активация: 7.6%
🟡 Gold   | Карт:  47 | Ср. окупаемость:  1.7 ходов | Активация: 15.8%
🔴 Red    | Карт: 101 | Ср. окупаемость:  6.1 ходов | Активация: 8.6%
🟣 Purple | Карт:  13 | Ср. окупаемость:  5.0 ходов | Активация: 0.9%
```

### 🔤 DSL-движок эффектов
| Команда                    | Описание                    | Пример                 |
|----------------------------|-----------------------------|------------------------|
| `GET <N>`                  | +N монет активатору         | `GET 3`                |
| `GETALL <N>`               | +N монет всем с этой картой | `GETALL 1`             |
| `STEAL_MONEY <TARGET> <N>` | Украсть N монет             | `STEAL_MONEY RANDOM 2` |
| `STEAL_CARD [TARGET]`      | Украсть карту из руки       | `STEAL_CARD RANDOM`    |
| `GETBY <COLOR> <MULT>`     | Доход за карты цвета        | `GETBY Blue 2`         |

## 🚀 Запуск проекта

### 📋 Требования
- .NET 8 SDK
- Файл `Cards.xlsx` в корне проекта
- Изображения карт в `wwwroot/images/cards/`

### ▶️ Быстрый старт
```bash
# 1. Клонирование и восстановление
git clone https://github.com/RelaxerR/TinyCityCardGame_online.git
cd TinyCityCardGame_online
dotnet restore

# 2. Запуск сервера
dotnet run

# 3. Игра
#    Хост: открыть http://localhost:5090 → создать комнату
#    Гость: ввести имя + код комнаты → присоединиться
```

### ⚙️ Настройка баланса (`appsettings.json`)
```json
{
  "GameSettings": {
    "WinTarget": 100,
    "DailyIncome": 1,
    "StartCoinsMin": 5,
    "StartCoinsMax": 10,
    "ColorChanceBlue": 50,
    "ColorChanceGold": 30, 
    "ColorChanceRed": 25,
    "ColorChancePurple": 5,
    "MarketSizeFormula": "{players_count} + 1",
    "MaxRounds": 100
  }
}
```

### ➕ Добавление новой карты
1. Откройте `Cards.xlsx`
2. Добавьте строку с полями:

| Колонка | Поле        | Пример                              |
|---------|-------------|-------------------------------------|
| A       | Name        | «Кофе-брейк от Искандера»           |
| B       | Color       | Blue                                |
| C       | Effect      | `GETALL 1`                          |
| D       | Cost        | 1                                   |
| E       | Reward      | 1                                   |
| F       | Icon        | `coffee.png`                        |
| G       | Description | «+1 корм всем с этой картой»        |
| H       | Weight      | 10                                  |
| I       | Narrative   | «Искандер принёс кофе после митапа» |

3. Поместите иконку в `wwwroot/images/cards/coffee.png`
4. Перезапустите сервер
